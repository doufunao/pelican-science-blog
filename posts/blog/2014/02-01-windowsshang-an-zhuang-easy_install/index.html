Windows上安装easy_install

Date 2014-02-01 22:46 Tags ['python' / 'easy_install' / 'setuptool']

第一步，在官网下载ez_setup.py文件https://pypi.python.org/pypi/setuptools#downloads，在命令行cmd里编译执行ez_setup.py

>python ez_setup.py

第二步，在环境变量path里增加个路径

C:\python33\scripts

第三步，利用easy_install安装BeautifulSoup4。在命令行cmd里输入

>easy_install beautifulsoup4
